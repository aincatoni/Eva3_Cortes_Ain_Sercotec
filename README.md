# Eva3 Cortes Ain Sercotec

Landing page para la evaluacion 3 de Desarrollo Frontend, basada en el Centro de Desarrollo de Negocios Sercotec Santiago.

## Stack Base

- `Next.js`
- `React`
- `TypeScript`
- `Tailwind CSS`

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
